1. Fundamentos de C++

¿Qué es C++?

C++ es un lenguaje de programación de propósito general creado por Bjarne Stroustrup en 1985 como una extensión del lenguaje C. Es un lenguaje compilado, de tipado estático y multiparadigma que soporta programación estructurada, orientada a objetos y genérica.

Características clave

  • Eficiencia: C++ permite un control detallado sobre los recursos del sistema.
  • Portabilidad: El código puede compilarse en múltiples plataformas.
  • Orientado a objetos: Soporta clases, herencia, polimorfismo, etc.
  • STL: La Biblioteca Estándar proporciona estructuras de datos y algoritmos potentes.

Tu primer programa en C++

El clásico "Hola Mundo" en C++:

hola_mundo.cpp 
#include <iostream>

int main() {
    // Imprime "Hola Mundo!" en la consola
    std::cout << "Hola Mundo!" << std::endl;
    return 0;
}

Explicación:

  1. #include <iostream>: Incluye la biblioteca estándar para operaciones de entrada/salida.
  2. int main(): Punto de entrada del programa. Debe devolver un entero.
  3. std::cout: Objeto para imprimir en la consola.
  4. <<: Operador de inserción para enviar datos a la salida.
  5. std::endl: Inserta un salto de línea y vacía el buffer.
  6. return 0;: Indica que el programa terminó correctamente.

Compilación y ejecución

Para compilar y ejecutar un programa en C++:

1

Escribe el código

Crea un archivo con extensión .cpp y escribe tu código.

2

Compila el programa

Usa un compilador como g++ (GCC) o clang++ (LLVM):

g++ hola_mundo.cpp -o hola_mundo
3

Ejecuta el programa

./hola_mundo

Deberías ver la salida: Hola Mundo!

2. Variables y Tipos de Datos

Tipos Fundamentales

C++ ofrece varios tipos de datos fundamentales:

Tipo Tamaño (bytes) Descripción Ejemplo
int 4 Entero con signo 42
float 4 Punto flotante de precisión simple 3.14f
double 8 Punto flotante de precisión doble 3.141592
char 1 Carácter 'A'
bool 1 Booleano true
void - Sin tipo -

Nota sobre tamaños

Los tamaños pueden variar según la arquitectura y el compilador. Puedes usar sizeof() para verificar el tamaño en tu sistema:

std::cout << "Tamaño de int: " << sizeof(int) << " bytes" << std::endl;

Declaración de Variables

Las variables se declaran con un tipo y un nombre:

int edad = 25;
double precio = 99.95;
char inicial = 'R';
bool es_valido = true;

Constantes

Usa const para valores que no deben cambiar:

const double PI = 3.1415926535;
const int DIAS_SEMANA = 7;

Auto (C++11)

Desde C++11, puedes usar auto para deducción de tipos:

auto nombre = "RataLife";  // deduce const char*
auto version = 1.2;       // deduce double
auto iteraciones = 10;    // deduce int

Precaución con auto

Usa auto con cuidado. Puede hacer el código menos legible si se usa en exceso o con expresiones complejas.

3. Estructuras de Control

Condicionales

C++ ofrece varias estructuras para controlar el flujo del programa:

condicionales.cpp 
// if-else
int edad = 18;
if (edad >= 18) {
    std::cout << "Mayor de edad" << std::endl;
} else {
    std::cout << "Menor de edad" << std::endl;
}

// switch-case
char opcion = 'B';
switch (opcion) {
    case 'A':
        std::cout << "Opción A" << std::endl;
        break;
    case 'B':
        std::cout << "Opción B" << std::endl;
        break;
    default:
        std::cout << "Opción desconocida" << std::endl;
}

Bucles

C++ proporciona tres tipos principales de bucles:

// for loop
for (int i = 0; i < 5; i++) {
    std::cout << "Iteración " << i << std::endl;
}

// while loop
int contador = 0;
while (contador < 5) {
    std::cout << "Contador: " << contador << std::endl;
    contador++;
}

// do-while loop
int num = 0;
do {
    std::cout << "Número: " << num << std::endl;
    num++;
} while (num < 5);

Control de bucles

  • break: Termina el bucle inmediatamente
  • continue: Salta a la siguiente iteración
  • return: Sale de la función actual

Operador Ternario

Una forma concisa de escribir condicionales simples:

int a = 10, b = 20;
int mayor = (a > b) ? a : b;
std::cout << "El mayor es: " << mayor << std::endl;

4. Funciones

Definición de Funciones

Las funciones permiten organizar el código en bloques reutilizables:

funciones.cpp 
// Declaración (prototipo)
int suma(int a, int b);

// Definición
int suma(int a, int b) {
    return a + b;
}

int main() {
    int resultado = suma(5, 3);
    std::cout << "5 + 3 = " << resultado << std::endl;
    return 0;
}

Parámetros por Valor y Referencia

C++ permite pasar parámetros por valor o por referencia:

// Por valor (copia)
void incrementar_valor(int x) {
    x++;
}

// Por referencia (original)
void incrementar_referencia(int &x) {
    x++;
}

int main() {
    int num = 5;
    incrementar_valor(num);    // num sigue siendo 5
    incrementar_referencia(num); // num ahora es 6
    return 0;
}

Funciones Recursivas

Una función que se llama a sí misma:

int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}

int main() {
    std::cout << "5! = " << factorial(5) << std::endl; // 120
    return 0;
}

Cuidado con la recursión

La recursión puede causar desbordamiento de pila (stack overflow) si no tiene una condición de salida adecuada o si se hace demasiado profunda.

Sobrecarga de Funciones

C++ permite múltiples funciones con el mismo nombre pero diferentes parámetros:

void mostrar(int x) {
    std::cout << "Entero: " << x << std::endl;
}

void mostrar(double x) {
    std::cout << "Double: " << x << std::endl;
}

void mostrar(const char* x) {
    std::cout << "Cadena: " << x << std::endl;
}

int main() {
    mostrar(5);        // Entero: 5
    mostrar(3.14);     // Double: 3.14
    mostrar("Hola");   // Cadena: Hola
    return 0;
}

5. Programación Orientada a Objetos

Clases y Objetos

Las clases son la base de la POO en C++:

clases.cpp 
class Persona {
private:
    std::string nombre;
    int edad;
    
public:
    // Constructor
    Persona(std::string n, int e) : nombre(n), edad(e) {}
    
    // Métodos
    void saludar() {
        std::cout << "Hola, soy " << nombre << " y tengo " << edad << " años." << std::endl;
    }
    
    // Getters y Setters
    void setEdad(int e) { edad = e; }
    int getEdad() const { return edad; }
};

int main() {
    Persona juan("Juan", 25);
    juan.saludar();
    juan.setEdad(26);
    std::cout << "Nueva edad: " << juan.getEdad() << std::endl;
    return 0;
}

Herencia

Una clase puede heredar propiedades y métodos de otra:

class Empleado : public Persona {
private:
    std::string puesto;
    double salario;
    
public:
    Empleado(std::string n, int e, std::string p, double s) 
        : Persona(n, e), puesto(p), salario(s) {}
    
    void mostrarInfo() {
        saludar();
        std::cout << "Puesto: " << puesto << ", Salario: $" << salario << std::endl;
    }
};

int main() {
    Empleado maria("Maria", 30, "Desarrolladora", 50000);
    maria.mostrarInfo();
    return 0;
}

Polimorfismo

Las funciones pueden comportarse diferente según el objeto que las llama:

class Animal {
public:
    virtual void hacerSonido() {
        std::cout << "Sonido genérico" << std::endl;
    }
};

class Perro : public Animal {
public:
    void hacerSonido() override {
        std::cout << "Guau guau!" << std::endl;
    }
};

class Gato : public Animal {
public:
    void hacerSonido() override {
        std::cout << "Miau!" << std::endl;
    }
};

int main() {
    Animal* animales[] = {new Perro(), new Gato()};
    
    for (Animal* animal : animales) {
        animal->hacerSonido();
        delete animal;
    }
    
    return 0;
}

Conceptos clave de POO

  • Encapsulamiento: Ocultar detalles internos (private/public)
  • Abstracción: Mostrar solo lo esencial
  • Herencia: Reutilizar código de clases padre
  • Polimorfismo: Mismo método, diferentes comportamientos

6. Plantillas

Funciones Genéricas

Las plantillas permiten crear funciones que trabajan con múltiples tipos:

plantillas_funciones.cpp 
template 
T maximo(T a, T b) {
    return (a > b) ? a : b;
}

int main() {
    std::cout << maximo(3, 7) << std::endl;       // 7
    std::cout << maximo(3.14, 2.71) << std::endl; // 3.14
    std::cout << maximo('a', 'z') << std::endl;   // z
    return 0;
}

Clases Genéricas

También podemos crear clases genéricas:

template 
class Contenedor {
private:
    T elemento;
    
public:
    Contenedor(T elem) : elemento(elem) {}
    
    T getElemento() { return elemento; }
};

int main() {
    Contenedor c1(42);
    Contenedor c2("Hola");
    
    std::cout << c1.getElemento() << std::endl; // 42
    std::cout << c2.getElemento() << std::endl; // Hola
    return 0;
}

STL y Plantillas

La STL (Standard Template Library) hace un uso intensivo de plantillas:

#include 
#include 

int main() {
    std::vector numeros = {5, 2, 8, 1, 9};
    
    // Ordenar el vector
    std::sort(numeros.begin(), numeros.end());
    
    // Mostrar elementos
    for (int num : numeros) {
        std::cout << num << " ";
    }
    // Salida: 1 2 5 8 9
    
    return 0;
}

Consejo sobre plantillas

Las definiciones de plantillas generalmente se colocan en archivos de encabezado (.hpp) porque el compilador necesita ver la implementación completa para instanciarlas.

7. STL - Biblioteca Estándar

Contenedores

La STL proporciona varios contenedores para almacenar datos:

Contenedor Descripción Ejemplo
vector Array dinámico std::vector<int> v = {1, 2, 3};
list Lista doblemente enlazada std::list<std::string> nombres;
map Diccionario ordenado std::map<std::string, int> edades;
set Colección de elementos únicos std::set<int> numeros;
queue Cola FIFO std::queue<int> cola;

Iteradores

Los iteradores permiten recorrer contenedores:

#include 
#include 

int main() {
    std::vector numeros = {1, 2, 3, 4, 5};
    
    // Iterador tradicional
    for (std::vector::iterator it = numeros.begin(); it != numeros.end(); ++it) {
        std::cout << *it << " ";
    }
    
    // Range-based for loop (C++11)
    for (int num : numeros) {
        std::cout << num << " ";
    }
    
    return 0;
}

Algoritmos

La STL incluye muchos algoritmos útiles:

#include 
#include 

int main() {
    std::vector v = {5, 2, 8, 1, 9};
    
    // Ordenar
    std::sort(v.begin(), v.end());
    
    // Buscar
    if (std::binary_search(v.begin(), v.end(), 8)) {
        std::cout << "8 encontrado" << std::endl;
    }
    
    // Contar
    int cincos = std::count(v.begin(), v.end(), 5);
    
    // Transformar
    std::transform(v.begin(), v.end(), v.begin(), [](int x) { return x * 2; });
    
    return 0;
}

STL moderna (C++11/14/17/20)

  • Smart pointers: unique_ptr, shared_ptr
  • Lambdas: Funciones anónimas
  • Tuplas: std::tuple
  • Nuevos contenedores: unordered_map, array

8. Temas Avanzados

Punteros y Memoria

C++ permite manejo directo de memoria:

punteros.cpp 
int main() {
    int* ptr = new int(42); // Asignar memoria dinámica
    std::cout << *ptr << std::endl; // Acceder al valor
    delete ptr; // Liberar memoria
    
    // Arrays dinámicos
    int* arr = new int[5]{1, 2, 3, 4, 5};
    delete[] arr;
    
    // Smart pointers (C++11)
    std::unique_ptr smartPtr = std::make_unique(10);
    std::cout << *smartPtr << std::endl;
    // No necesita delete - se libera automáticamente
    
    return 0;
}

Excepciones

Manejo de errores con excepciones:

#include 

double dividir(double a, double b) {
    if (b == 0) {
        throw std::runtime_error("División por cero");
    }
    return a / b;
}

int main() {
    try {
        double resultado = dividir(10, 0);
        std::cout << resultado << std::endl;
    } catch (const std::exception& e) {
        std::cerr << "Error: " << e.what() << std::endl;
    }
    return 0;
}

Lambdas (C++11)

Funciones anónimas útiles para algoritmos:

#include 
#include 

int main() {
    std::vector numeros = {5, 2, 8, 1, 9};
    
    // Ordenar descendente con lambda
    std::sort(numeros.begin(), numeros.end(), [](int a, int b) {
        return a > b;
    });
    
    // Capturar variables locales
    int limite = 5;
    auto mayores = std::count_if(numeros.begin(), numeros.end(), [limite](int x) {
        return x > limite;
    });
    
    std::cout << "Números mayores que " << limite << ": " << mayores << std::endl;
    return 0;
}
