1. Fundamentos de Windows Forms

¿Qué es Windows Forms?

Windows Forms es un framework para crear aplicaciones de escritorio para Windows utilizando C# y .NET. Proporciona una amplia gama de controles visuales que permiten crear interfaces de usuario ricas y funcionales.

Características clave

  • Basado en controles: Botones, cuadros de texto, listas, etc.
  • Diseño visual: Puedes diseñar formularios con el diseñador de Visual Studio
  • Modelo de eventos: Responde a acciones del usuario como clicks y teclas
  • Integración con .NET: Acceso completo a las bibliotecas de .NET
  • Arquitectura MVC: Separa lógica de negocio de la interfaz de usuario

Tu primer proyecto en Windows Forms

Creación de un formulario básico con un botón:

Program.cs 
using System;
using System.Windows.Forms;

namespace MiPrimerFormulario
{
    static class Program
    {
        [STAThread]
        static void Main()
        {
            Application.EnableVisualStyles();
            Application.SetCompatibleTextRenderingDefault(false);
            Application.Run(new Form1());
        }
    }
}
Form1.cs 
using System;
using System.Windows.Forms;

namespace MiPrimerFormulario
{
    public partial class Form1 : Form
    {
        public Form1()
        {
            InitializeComponent();
            
            // Crear un botón programáticamente
            Button btnSaludo = new Button();
            btnSaludo.Text = "Saludar";
            btnSaludo.Location = new System.Drawing.Point(50, 50);
            btnSaludo.Click += BtnSaludo_Click;
            
            this.Controls.Add(btnSaludo);
        }

        private void BtnSaludo_Click(object sender, EventArgs e)
        {
            MessageBox.Show("¡Hola desde Windows Forms!");
        }
    }
}

Explicación:

  1. Application.Run(new Form1()): Inicia la aplicación mostrando Form1.
  2. InitializeComponent(): Método generado por el diseñador para inicializar componentes.
  3. Button: Control que representa un botón clickeable.
  4. Click += BtnSaludo_Click: Asocia el evento Click con el método manejador.
  5. MessageBox.Show: Muestra un cuadro de diálogo con un mensaje.

Creación de un proyecto

Para crear un proyecto de Windows Forms:

1

Crea un proyecto

En Visual Studio, selecciona "Nuevo proyecto" y elige "Aplicación de Windows Forms (.NET Framework)" o "Aplicación de Windows Forms (.NET)"

2

Diseña el formulario

Usa el diseñador visual para agregar controles desde el cuadro de herramientas.

3

Escribe el código

Haz doble clic en controles para generar manejadores de eventos o escribe código en los archivos .cs.

4

Ejecuta el programa

Presiona F5 o haz clic en "Iniciar" para ejecutar tu aplicación.

Estructura de un proyecto

Un proyecto típico de Windows Forms contiene:

Archivo/Carpeta Descripción
Program.cs Punto de entrada de la aplicación
Form1.cs Código del formulario principal
Form1.Designer.cs Código generado por el diseñador
Form1.resx Recursos del formulario (imágenes, cadenas)
Properties/ Configuración de la aplicación (icono, versión)
References/ Ensamblados referenciados

Ejemplo: Formulario de Registro

Creación de un formulario con múltiples controles:

RegistroForm.cs 
public partial class RegistroForm : Form
{
    public RegistroForm()
    {
        InitializeComponent();
        
        // Configurar controles
        txtNombre.Text = "Ingrese su nombre";
        txtNombre.ForeColor = Color.Gray;
        
        dtpNacimiento.Value = DateTime.Now.AddYears(-18);
        dtpNacimiento.MaxDate = DateTime.Now;
        
        // Llenar ComboBox
        cmbPais.Items.AddRange(new string[] { "España", "México", "Argentina", "Colombia" });
        cmbPais.SelectedIndex = 0;
        
        // Configurar botón
        btnRegistrar.Enabled = false;
    }

    private void txtNombre_Enter(object sender, EventArgs e)
    {
        if (txtNombre.Text == "Ingrese su nombre")
        {
            txtNombre.Text = "";
            txtNombre.ForeColor = Color.Black;
        }
    }

    private void txtNombre_Leave(object sender, EventArgs e)
    {
        if (string.IsNullOrWhiteSpace(txtNombre.Text))
        {
            txtNombre.Text = "Ingrese su nombre";
            txtNombre.ForeColor = Color.Gray;
        }
    }

    private void txtNombre_TextChanged(object sender, EventArgs e)
    {
        btnRegistrar.Enabled = !string.IsNullOrWhiteSpace(txtNombre.Text) && 
                              txtNombre.Text != "Ingrese su nombre";
    }

    private void btnRegistrar_Click(object sender, EventArgs e)
    {
        string mensaje = $"Nombre: {txtNombre.Text}\n" +
                        $"Fecha Nacimiento: {dtpNacimiento.Value.ToShortDateString()}\n" +
                        $"País: {cmbPais.SelectedItem}\n" +
                        $"Acepta términos: {(chkTerminos.Checked ? "Sí" : "No")}";
        
        MessageBox.Show(mensaje, "Datos Registrados", 
                      MessageBoxButtons.OK, MessageBoxIcon.Information);
    }
}

Funcionalidades implementadas:

  • Placeholder en TextBox: Muestra texto gris cuando está vacío.
  • Validación de entrada: Habilita el botón solo cuando hay texto válido.
  • Interacción entre controles: Los cambios en un control afectan a otros.
  • Mensaje de confirmación: Muestra los datos ingresados al hacer clic.

3. Manejo de Eventos

Modelo de Eventos en Windows Forms

Los eventos permiten que tu aplicación responda a acciones del usuario o del sistema:

Conceptos clave

  • Publicador: El control que genera el evento (ej. Button)
  • Suscriptor: El método que maneja el evento
  • Delegado: Define la firma del método manejador
  • EventArgs: Clase que contiene datos sobre el evento

Eventos Comunes

Evento Control Típico Descripción
Click Button, Form Ocurre al hacer clic en el control
TextChanged TextBox, ComboBox Ocurre cuando cambia el texto
CheckedChanged CheckBox, RadioButton Ocurre cuando cambia el estado
SelectedIndexChanged ComboBox, ListBox Ocurre cuando cambia la selección
Enter/Leave Cualquier control Ocurre cuando el control gana/pierde el foco
KeyDown/KeyUp Cualquier control Ocurre al presionar/soltar una tecla
MouseEnter/MouseLeave Cualquier control Ocurre cuando el mouse entra/sale del control
FormClosing Form Ocurre antes de que el formulario se cierre
Load Form Ocurre cuando el formulario se carga

Manejo de Eventos

Hay varias formas de manejar eventos en Windows Forms:

EventosForm.cs 
public partial class EventosForm : Form
{
    public EventosForm()
    {
        InitializeComponent();
        
        // 1. Usando el diseñador (genera automáticamente)
        btnDesigner.Click += btnDesigner_Click;
        
        // 2. Método anónimo
        btnAnonimo.Click += delegate (object sender, EventArgs e)
        {
            MessageBox.Show("Manejador anónimo");
        };
        
        // 3. Expresión lambda
        btnLambda.Click += (s, e) => 
        {
            MessageBox.Show("Expresión lambda");
        };
        
        // 4. Múltiples controles con el mismo manejador
        btnMulti1.Click += BtnMulti_Click;
        btnMulti2.Click += BtnMulti_Click;
        
        // 5. Evento personalizado
        MiControlPersonalizado.MiEventoPersonalizado += (s, e) =>
        {
            MessageBox.Show($"Evento personalizado: {e.MiDato}");
        };
    }
    
    // 1. Manejador generado por el diseñador
    private void btnDesigner_Click(object sender, EventArgs e)
    {
        MessageBox.Show("Manejador del diseñador");
    }
    
    // 4. Manejador compartido
    private void BtnMulti_Click(object sender, EventArgs e)
    {
        Button btn = (Button)sender;
        MessageBox.Show($"Click en {btn.Name}");
    }
}

// Clase con evento personalizado
public class MiControlPersonalizado : Control
{
    public event EventHandler<MiEventArgs> MiEventoPersonalizado;
    
    protected virtual void OnMiEventoPersonalizado(string dato)
    {
        MiEventoPersonalizado?.Invoke(this, new MiEventArgs(dato));
    }
    
    // Lógica que dispara el evento...
}

public class MiEventArgs : EventArgs
{
    public string MiDato { get; }
    
    public MiEventArgs(string dato)
    {
        MiDato = dato;
    }
}

Mejores prácticas para eventos

  • Usa nombres descriptivos para los manejadores (ej. btnAceptar_Click)
  • Desuscribe eventos cuando ya no sean necesarios para evitar memory leaks
  • Para eventos compartidos, usa el parámetro sender para identificar el control
  • Mantén el código en manejadores corto, delega lógica compleja a otros métodos

Eventos de Teclado y Ratón

Ejemplo avanzado de manejo de eventos de entrada:

InputEventsForm.cs 
public partial class InputEventsForm : Form
{
    private Point startPoint;
    private bool isDrawing = false;
    
    public InputEventsForm()
    {
        InitializeComponent();
        
        // Configurar eventos
        this.KeyPreview = true; // Recibir eventos de teclado del formulario
        
        pnlDibujo.MouseDown += PnlDibujo_MouseDown;
        pnlDibujo.MouseMove += PnlDibujo_MouseMove;
        pnlDibujo.MouseUp += PnlDibujo_MouseUp;
        this.KeyDown += InputEventsForm_KeyDown;
    }
    
    private void PnlDibujo_MouseDown(object sender, MouseEventArgs e)
    {
        if (e.Button == MouseButtons.Left)
        {
            isDrawing = true;
            startPoint = e.Location;
            
            // Crear un nuevo "lápiz"
            Graphics g = pnlDibujo.CreateGraphics();
            g.FillEllipse(Brushes.Black, e.X - 2, e.Y - 2, 4, 4);
            g.Dispose();
        }
    }
    
    private void PnlDibujo_MouseMove(object sender, MouseEventArgs e)
    {
        if (isDrawing)
        {
            Graphics g = pnlDibujo.CreateGraphics();
            g.DrawLine(Pens.Black, startPoint, e.Location);
            g.Dispose();
            startPoint = e.Location;
        }
    }
    
    private void PnlDibujo_MouseUp(object sender, MouseEventArgs e)
    {
        isDrawing = false;
    }
    
    private void InputEventsForm_KeyDown(object sender, KeyEventArgs e)
    {
        switch (e.KeyCode)
        {
            case Keys.Escape:
                pnlDibujo.Invalidate(); // Limpiar el panel
                break;
            case Keys.S when e.Control:
                // Ctrl+S para guardar
                GuardarDibujo();
                break;
        }
    }
    
    private void GuardarDibujo()
    {
        Bitmap bmp = new Bitmap(pnlDibujo.Width, pnlDibujo.Height);
        pnlDibujo.DrawToBitmap(bmp, pnlDibujo.ClientRectangle);
        
        SaveFileDialog sfd = new SaveFileDialog();
        sfd.Filter = "Imagen PNG|*.png|Imagen JPEG|*.jpg";
        
        if (sfd.ShowDialog() == DialogResult.OK)
        {
            ImageFormat formato = Path.GetExtension(sfd.FileName) == ".png" ? 
                               ImageFormat.Png : ImageFormat.Jpeg;
            bmp.Save(sfd.FileName, formato);
        }
    }
}

Uso de Diálogos

Ejemplos prácticos de diálogos comunes:

DialogosForm.cs 
public partial class DialogosForm : Form
{
    public DialogosForm()
    {
        InitializeComponent();
    }
    
    // MessageBox
    private void btnMensaje_Click(object sender, EventArgs e)
    {
        DialogResult result = MessageBox.Show(
            "¿Está seguro de eliminar este registro?",
            "Confirmación",
            MessageBoxButtons.YesNoCancel,
            MessageBoxIcon.Warning,
            MessageBoxDefaultButton.Button2);
            
        if (result == DialogResult.Yes)
        {
            // Lógica para eliminar
            MessageBox.Show("Registro eliminado", "Éxito", 
                          MessageBoxButtons.OK, MessageBoxIcon.Information);
        }
        else if (result == DialogResult.No)
        {
            // Otra acción
        }
    }
    
    // OpenFileDialog
    private void btnAbrir_Click(object sender, EventArgs e)
    {
        OpenFileDialog ofd = new OpenFileDialog();
        ofd.Title = "Seleccione un archivo de imagen";
        ofd.Filter = "Imágenes|*.jpg;*.png;*.bmp|Todos los archivos|*.*";
        ofd.Multiselect = true;
        
        if (ofd.ShowDialog() == DialogResult.OK)
        {
            foreach (string archivo in ofd.FileNames)
            {
                lstArchivos.Items.Add(archivo);
            }
        }
    }
    
    // SaveFileDialog
    private void btnGuardar_Click(object sender, EventArgs e)
    {
        SaveFileDialog sfd = new SaveFileDialog();
        sfd.Filter = "Archivo de texto|*.txt|Documento Word|*.docx";
        sfd.DefaultExt = "txt";
        sfd.AddExtension = true;
        sfd.OverwritePrompt = true;
        
        if (sfd.ShowDialog() == DialogResult.OK)
        {
            File.WriteAllText(sfd.FileName, txtContenido.Text);
        }
    }
    
    // ColorDialog y FontDialog
    private void btnEstilo_Click(object sender, EventArgs e)
    {
        ColorDialog cd = new ColorDialog();
        cd.Color = txtEjemplo.ForeColor;
        
        if (cd.ShowDialog() == DialogResult.OK)
        {
            txtEjemplo.ForeColor = cd.Color;
        }
        
        FontDialog fd = new FontDialog();
        fd.Font = txtEjemplo.Font;
        fd.ShowColor = false; // Ya configuramos el color
        
        if (fd.ShowDialog() == DialogResult.OK)
        {
            txtEjemplo.Font = fd.Font;
        }
    }
}

Formularios Personalizados como Diálogos

Puedes crear tus propios diálogos personalizados:

LoginDialog.cs 
public partial class LoginDialog : Form
{
    public string Usuario { get; private set; }
    public string Contrasena { get; private set; }
    
    public LoginDialog()
    {
        InitializeComponent();
        this.DialogResult = DialogResult.Cancel; // Por defecto es Cancel
    }
    
    private void btnAceptar_Click(object sender, EventArgs e)
    {
        if (string.IsNullOrWhiteSpace(txtUsuario.Text) || 
            string.IsNullOrWhiteSpace(txtContrasena.Text))
        {
            MessageBox.Show("Debe ingresar usuario y contraseña", "Error", 
                          MessageBoxButtons.OK, MessageBoxIcon.Error);
            return;
        }
        
        Usuario = txtUsuario.Text;
        Contrasena = txtContrasena.Text;
        this.DialogResult = DialogResult.OK;
        this.Close();
    }
    
    private void btnCancelar_Click(object sender, EventArgs e)
    {
        this.Close();
    }
}

// Uso desde otro formulario:
private void btnLogin_Click(object sender, EventArgs e)
{
    LoginDialog login = new LoginDialog();
    
    if (login.ShowDialog() == DialogResult.OK)
    {
        // Verificar credenciales
        if (Autenticar(login.Usuario, login.Contrasena))
        {
            MessageBox.Show("Bienvenido!", "Éxito");
        }
        else
        {
            MessageBox.Show("Credenciales incorrectas", "Error");
        }
    }
    else
    {
        MessageBox.Show("Inicio de sesión cancelado", "Información");
    }
    
    login.Dispose();
}

Mejores prácticas para diálogos

  • Usa ShowDialog() para diálogos modales que deben completarse
  • Configura propiedades como Title, Filter para mejor UX
  • Verifica siempre el DialogResult antes de procesar
  • Para diálogos personalizados, expone propiedades para acceder a los datos
  • Implementa validación en diálogos de entrada

Diseño Responsivo

Técnicas para crear formularios que se adaptan a diferentes tamaños:

ResponsiveForm.cs 
public partial class ResponsiveForm : Form
{
    private TableLayoutPanel mainTable;
    private Button btnEjemplo;
    private TextBox txtEjemplo;
    
    public ResponsiveForm()
    {
        InitializeComponent();
        CrearControlesProgramaticamente();
        ConfigurarDiseñoResponsivo();
    }
    
    private void CrearControlesProgramaticamente()
    {
        // Configurar TableLayoutPanel principal
        mainTable = new TableLayoutPanel();
        mainTable.Dock = DockStyle.Fill;
        mainTable.ColumnCount = 3;
        mainTable.RowCount = 4;
        mainTable.ColumnStyles.Add(new ColumnStyle(SizeType.Percent, 20));
        mainTable.ColumnStyles.Add(new ColumnStyle(SizeType.Percent, 60));
        mainTable.ColumnStyles.Add(new ColumnStyle(SizeType.Percent, 20));
        
        for (int i = 0; i < mainTable.RowCount; i++)
        {
            mainTable.RowStyles.Add(new RowStyle(SizeType.AutoSize));
        }
        
        // Crear controles
        btnEjemplo = new Button();
        btnEjemplo.Text = "Botón Responsivo";
        btnEjemplo.Dock = DockStyle.Fill;
        btnEjemplo.Margin = new Padding(5);
        
        txtEjemplo = new TextBox();
        txtEjemplo.Dock = DockStyle.Fill;
        txtEjemplo.Multiline = true;
        txtEjemplo.ScrollBars = ScrollBars.Vertical;
        txtEjemplo.Margin = new Padding(5);
        
        // Agregar controles al TableLayoutPanel
        mainTable.Controls.Add(btnEjemplo, 1, 1);
        mainTable.Controls.Add(txtEjemplo, 1, 2);
        
        // Hacer que el TextBox ocupe 2 filas
        txtEjemplo.Anchor = AnchorStyles.Top | AnchorStyles.Bottom | 
                           AnchorStyles.Left | AnchorStyles.Right;
        
        // Agregar TableLayoutPanel al formulario
        this.Controls.Add(mainTable);
    }
    
    private void ConfigurarDiseñoResponsivo()
    {
        // Configurar el formulario
        this.Text = "Formulario Responsivo";
        this.MinimumSize = new Size(300, 200);
        this.StartPosition = FormStartPosition.CenterScreen;
        
        // Manejar el evento Resize
        this.Resize += ResponsiveForm_Resize;
    }
    
    private void ResponsiveForm_Resize(object sender, EventArgs e)
    {
        // Ajustar tamaño de fuente según el tamaño del formulario
        if (this.Width < 500)
        {
            btnEjemplo.Font = new Font(btnEjemplo.Font.FontFamily, 8);
        }
        else
        {
            btnEjemplo.Font = new Font(btnEjemplo.Font.FontFamily, 10);
        }
        
        // Ajustar número de columnas/rows según orientación
        if (this.ClientSize.Width > this.ClientSize.Height)
        {
            // Modo horizontal
            mainTable.ColumnCount = 3;
            mainTable.RowCount = 2;
            mainTable.SetColumnSpan(txtEjemplo, 2);
        }
        else
        {
            // Modo vertical
            mainTable.ColumnCount = 1;
            mainTable.RowCount = 4;
            mainTable.SetColumnSpan(txtEjemplo, 1);
        }
    }
}

Controles de Diseño Avanzado

Windows Forms ofrece controles especializados para diseño complejo:

AdvancedLayoutForm.cs 
public partial class AdvancedLayoutForm : Form
{
    public AdvancedLayoutForm()
    {
        InitializeComponent();
        ConfigurarSplitContainer();
        ConfigurarTabControl();
        ConfigurarFlowLayout();
    }
    
    private void ConfigurarSplitContainer()
    {
        // SplitContainer horizontal con proporción 30/70
        SplitContainer splitMain = new SplitContainer();
        splitMain.Dock = DockStyle.Fill;
        splitMain.Orientation = Orientation.Horizontal;
        splitMain.SplitterDistance = (int)(this.ClientSize.Height * 0.3);
        splitMain.FixedPanel = FixedPanel.Panel1;
        
        // Panel superior (30%)
        TreeView treeView = new TreeView();
        treeView.Dock = DockStyle.Fill;
        treeView.Nodes.Add("Elemento 1");
        treeView.Nodes.Add("Elemento 2");
        splitMain.Panel1.Controls.Add(treeView);
        
        // Panel inferior (70%)
        SplitContainer splitSecondary = new SplitContainer();
        splitSecondary.Dock = DockStyle.Fill;
        splitSecondary.SplitterDistance = (int)(this.ClientSize.Width * 0.2);
        
        // Panel izquierdo (20% del espacio restante)
        ListBox listBox = new ListBox();
        listBox.Dock = DockStyle.Fill;
        listBox.Items.AddRange(new string[] { "Opción A", "Opción B", "Opción C" });
        splitSecondary.Panel1.Controls.Add(listBox);
        
        // Panel derecho (80% del espacio restante)
        PropertyGrid propGrid = new PropertyGrid();
        propGrid.Dock = DockStyle.Fill;
        propGrid.SelectedObject = this;
        splitSecondary.Panel2.Controls.Add(propGrid);
        
        splitMain.Panel2.Controls.Add(splitSecondary);
        this.Controls.Add(splitMain);
    }
    
    private void ConfigurarTabControl()
    {
        TabControl tabControl = new TabControl();
        tabControl.Dock = DockStyle.Fill;
        
        // Pestaña 1
        TabPage tabPage1 = new TabPage("Configuración");
        tabPage1.Controls.Add(new PropertyGrid() { Dock = DockStyle.Fill });
        
        // Pestaña 2
        TabPage tabPage2 = new TabPage("Documentación");
        WebBrowser browser = new WebBrowser();
        browser.Dock = DockStyle.Fill;
        browser.Navigate("https://docs.microsoft.com");
        tabPage2.Controls.Add(browser);
        
        tabControl.TabPages.Add(tabPage1);
        tabControl.TabPages.Add(tabPage2);
        
        // Agregar a un panel existente
        pnlContenedor.Controls.Add(tabControl);
    }
    
    private void ConfigurarFlowLayout()
    {
        FlowLayoutPanel flowPanel = new FlowLayoutPanel();
        flowPanel.Dock = DockStyle.Bottom;
        flowPanel.Height = 100;
        flowPanel.FlowDirection = FlowDirection.LeftToRight;
        flowPanel.WrapContents = true;
        flowPanel.AutoScroll = true;
        
        for (int i = 1; i <= 20; i++)
        {
            Button btn = new Button();
            btn.Text = $"Botón {i}";
            btn.Margin = new Padding(5);
            btn.Width = 80;
            flowPanel.Controls.Add(btn);
        }
        
        this.Controls.Add(flowPanel);
    }
}

Consejos para diseño efectivo

  • Usa Dock y Anchor para diseños flexibles
  • Agrupa controles relacionados con Panel o GroupBox
  • Utiliza TableLayoutPanel para alineación precisa
  • Considera el flujo de trabajo del usuario al organizar controles
  • Mantén consistencia en tamaños, colores y fuentes
  • Prueba tu interfaz en diferentes resoluciones

DatabaseForm.cs 
public partial class DatabaseForm : Form
{
    private SqlConnection connection;
    private SqlDataAdapter adapter;
    private DataTable table;
    private BindingSource bindingSource = new BindingSource();
    
    public DatabaseForm()
    {
        InitializeComponent();
        ConfigurarConexion();
        CargarDatos();
    }
    
    private void ConfigurarConexion()
    {
        // Cadena de conexión (normalmente se guarda en configuración)
        string connectionString = "Server=localhost;Database=MiBaseDatos;Integrated Security=True;";
        connection = new SqlConnection(connectionString);
        
        // Configurar DataAdapter con comandos CRUD
        adapter = new SqlDataAdapter();
        
        // Select
        adapter.SelectCommand = new SqlCommand("SELECT * FROM Productos", connection);
        
        // Insert
        adapter.InsertCommand = new SqlCommand(
            "INSERT INTO Productos (Nombre, Precio, Stock) VALUES (@Nombre, @Precio, @Stock)", 
            connection);
        adapter.InsertCommand.Parameters.Add("@Nombre", SqlDbType.NVarChar, 50, "Nombre");
        adapter.InsertCommand.Parameters.Add("@Precio", SqlDbType.Decimal, 0, "Precio");
        adapter.InsertCommand.Parameters.Add("@Stock", SqlDbType.Int, 0, "Stock");
        
        // Update
        adapter.UpdateCommand = new SqlCommand(
            "UPDATE Productos SET Nombre = @Nombre, Precio = @Precio, Stock = @Stock WHERE Id = @Id", 
            connection);
        adapter.UpdateCommand.Parameters.Add("@Nombre", SqlDbType.NVarChar, 50, "Nombre");
        adapter.UpdateCommand.Parameters.Add("@Precio", SqlDbType.Decimal, 0, "Precio");
        adapter.UpdateCommand.Parameters.Add("@Stock", SqlDbType.Int, 0, "Stock");
        adapter.UpdateCommand.Parameters.Add("@Id", SqlDbType.Int, 0, "Id");
        
        // Delete
        adapter.DeleteCommand = new SqlCommand(
            "DELETE FROM Productos WHERE Id = @Id", 
            connection);
        adapter.DeleteCommand.Parameters.Add("@Id", SqlDbType.Int, 0, "Id");
    }
    
    private void CargarDatos()
    {
        try
        {
            table = new DataTable();
            adapter.Fill(table);
            
            bindingSource.DataSource = table;
            dgvProductos.DataSource = bindingSource;
            
            // Configurar formato
            dgvProductos.Columns["Id"].Visible = false;
            dgvProductos.Columns["Precio"].DefaultCellStyle.Format = "C2";
        }
        catch (Exception ex)
        {
            MessageBox.Show($"Error al cargar datos: {ex.Message}", "Error");
        }
    }
    
    private void btnGuardar_Click(object sender, EventArgs e)
    {
        try
        {
            bindingSource.EndEdit();
            adapter.Update(table);
            MessageBox.Show("Cambios guardados correctamente", "Éxito");
        }
        catch (Exception ex)
        {
            MessageBox.Show($"Error al guardar: {ex.Message}", "Error");
        }
    }
    
    private void btnBuscar_Click(object sender, EventArgs e)
    {
        string filtro = txtBusqueda.Text.Trim();
        
        if (!string.IsNullOrEmpty(filtro))
        {
            bindingSource.Filter = $"Nombre LIKE '%{filtro}%'";
        }
        else
        {
            bindingSource.Filter = "";
        }
    }
    
    private void DatabaseForm_FormClosing(object sender, FormClosingEventArgs e)
    {
        if (connection.State != ConnectionState.Closed)
        {
            connection.Close();
        }
    }
}

Hilos y Operaciones Asíncronas

Manejo de operaciones largas sin bloquear la interfaz:

AsyncForm.cs 
public partial class AsyncForm : Form
{
    public AsyncForm()
    {
        InitializeComponent();
    }
    
    // Método sincrónico (bloquea la UI)
    private void btnSync_Click(object sender, EventArgs e)
    {
        lblEstado.Text = "Procesando...";
        this.Cursor = Cursors.WaitCursor;
        
        // Simular trabajo largo
        ProcesamientoLargo();
        
        lblEstado.Text = "Completado";
        this.Cursor = Cursors.Default;
    }
    
    // Método usando BackgroundWorker (patrón antiguo)
    private void btnBackgroundWorker_Click(object sender, EventArgs e)
    {
        BackgroundWorker worker = new BackgroundWorker();
        worker.WorkerReportsProgress = true;
        
        worker.DoWork += (s, args) =>
        {
            for (int i = 1; i <= 10; i++)
            {
                Thread.Sleep(500); // Trabajo simulado
                worker.ReportProgress(i * 10);
            }
        };
        
        worker.ProgressChanged += (s, args) =>
        {
            progressBar.Value = args.ProgressPercentage;
            lblProgreso.Text = $"{args.ProgressPercentage}%";
        };
        
        worker.RunWorkerCompleted += (s, args) =>
        {
            lblEstado.Text = "Completado con BackgroundWorker";
        };
        
        lblEstado.Text = "Procesando con BackgroundWorker...";
        worker.RunWorkerAsync();
    }
    
    // Método asíncrono moderno (async/await)
    private async void btnAsync_Click(object sender, EventArgs e)
    {
        lblEstado.Text = "Procesando asíncronamente...";
        this.Cursor = Cursors.WaitCursor;
        btnAsync.Enabled = false;
        
        try
        {
            // Ejecutar tarea en segundo plano
            await Task.Run(() => ProcesamientoLargo());
            
            lblEstado.Text = "Completado asíncronamente";
        }
        catch (Exception ex)
        {
            lblEstado.Text = $"Error: {ex.Message}";
        }
        finally
        {
            this.Cursor = Cursors.Default;
            btnAsync.Enabled = true;
        }
    }
    
    private void ProcesamientoLargo()
    {
        // Simular trabajo que toma tiempo
        Thread.Sleep(3000);
    }
    
    // Ejemplo con progreso y cancelación
    private async void btnProgreso_Click(object sender, EventArgs e)
    {
        var progress = new Progress<int>(porcentaje =>
        {
            progressBar.Value = porcentaje;
            lblProgreso.Text = $"{porcentaje}%";
        });
        
        var cancellationTokenSource = new CancellationTokenSource();
        btnCancelar.Enabled = true;
        btnCancelar.Click += (s, args) => cancellationTokenSource.Cancel();
        
        try
        {
            await Task.Run(() => ProcesamientoConProgreso(progress, cancellationTokenSource.Token));
            lblEstado.Text = "Proceso completado";
        }
        catch (OperationCanceledException)
        {
            lblEstado.Text = "Proceso cancelado";
        }
        finally
        {
            btnCancelar.Enabled = false;
        }
    }
    
    private void ProcesamientoConProgreso(IProgress<int> progress, CancellationToken token)
    {
        for (int i = 0; i <= 100; i++)
        {
            token.ThrowIfCancellationRequested();
            Thread.Sleep(50); // Trabajo simulado
            progress.Report(i);
        }
    }
}
