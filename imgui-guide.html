1. Introducción a Dear ImGui

¿Qué es Dear ImGui?

Dear ImGui (Immediate Mode Graphical User Interface) es una biblioteca ligera para C++ que permite crear interfaces gráficas de usuario de manera rápida y sencilla. A diferencia de los sistemas GUI tradicionales, ImGui sigue un paradigma de modo inmediato, lo que significa que la interfaz se reconstruye completamente en cada fotograma.

Características principales

  • Modo inmediato: Más simple que el modo retenido tradicional.
  • Portátil: Funciona en múltiples plataformas y backends.
  • Sin dependencias: Solo requiere un backend de renderizado.
  • Personalizable: Estilos y temas completamente configurables.
  • Eficiente: Optimizado para herramientas y aplicaciones en tiempo real.

Casos de uso típicos

Dear ImGui es ideal para:

Herramientas de desarrollo

Depuradores, editores de niveles, inspectores de variables.

Juegos

Interfaces de debug, menús de desarrollo.

Visualización de datos

Gráficos, métricas en tiempo real.

Aplicaciones con controles

Editores de parámetros, paneles de control.

No para todo

ImGui no está diseñado para interfaces de usuario complejas orientadas al usuario final. Es ideal para herramientas y desarrollo, pero no reemplaza a bibliotecas como Qt o wxWidgets para aplicaciones generales.

2. Configuración del Entorno

Obteniendo Dear ImGui

Puedes descargar Dear ImGui de varias formas:

1

Desde GitHub (recomendado)

Clona el repositorio oficial:

git clone https://github.com/ocornut/imgui.git
2

Como submodulo

Si usas git en tu proyecto:

git submodule add https://github.com/ocornut/imgui.git
3

Descarga manual

Descarga el .zip desde las releases.

Estructura del proyecto

La estructura básica de ImGui es simple:

  • imgui/
    • imconfig.h
    • imgui.h
    • imgui.cpp
    • imgui_demo.cpp
    • imgui_draw.cpp
    • imgui_widgets.cpp
    • backends/
      • imgui_impl_glfw.h
      • imgui_impl_glfw.cpp
      • imgui_impl_opengl3.h
      • imgui_impl_opengl3.cpp
      • ... otros backends

Integración con un proyecto

Para integrar ImGui en tu proyecto:

  1. Añade los archivos principales a tu proyecto (imgui.cpp, imgui_draw.cpp, imgui_widgets.cpp).
  2. Incluye imgui.h donde lo necesites.
  3. Elige un backend de renderizado (GLFW+OpenGL, SDL, DirectX, etc.).
  4. Configura el bucle principal para manejar ImGui.

Backends disponibles

ImGui necesita un backend para renderizado y manejo de entrada. Los más comunes:

Backend Plataforma Renderizador
GLFW Multiplataforma OpenGL, Vulkan
SDL Multiplataforma OpenGL, Direct3D
Win32 Windows DirectX
Allegro Multiplataforma OpenGL

Consejo

Para comenzar rápidamente, usa el combo GLFW + OpenGL3, que es multiplataforma y fácil de configurar.

3. Creando tu Primera Ventana

Estructura básica de una aplicación con ImGui

Toda aplicación con ImGui sigue un flujo similar:

1

Inicialización

Configura el contexto de ImGui y los backends necesarios.

2

Bucle principal

Donde se manejan los frames de ImGui y se renderiza la interfaz.

3

Limpieza

Liberación de recursos al finalizar la aplicación.

Tu primera ventana

Veamos cómo crear una ventana simple con un botón:

Ventana Básica 
// Dentro de tu bucle principal
ImGui::Begin("Mi Primera Ventana");
ImGui::Text("¡Hola desde ImGui!");
if (ImGui::Button("Presióname")) {
    // Lógica cuando se presiona el botón
    std::cout << "Botón presionado!\n";
}
ImGui::End();

¿Cómo funciona?

  • ImGui::Begin() inicia una nueva ventana.
  • ImGui::Text() muestra texto estático.
  • ImGui::Button() crea un botón clickeable.
  • ImGui::End() cierra la ventana.

Manejando el estado

ImGui no maneja estado interno, así que debes gestionarlo tú:

// Variables de estado (deben persistir entre frames)
static bool show_window = true;
static int counter = 0;

if (show_window) {
    ImGui::Begin("Ventana con Estado", &show_window);
    ImGui::Text("Contador: %d", counter);
    if (ImGui::Button("Incrementar")) {
        counter++;
    }
    ImGui::End();
}

Consejo importante

Usa variables static o variables miembro de clase para mantener el estado entre frames. Las variables locales se perderán en cada iteración del bucle.

Ventanas modales

Puedes crear ventanas modales (que bloquean la interacción con otras ventanas):

if (ImGui::Button("Mostrar Modal")) {
    ImGui::OpenPopup("Mi Modal");
}

if (ImGui::BeginPopupModal("Mi Modal", NULL, ImGuiWindowFlags_AlwaysAutoResize)) {
    ImGui::Text("¡Esto es un diálogo modal!");
    if (ImGui::Button("Cerrar")) {
        ImGui::CloseCurrentPopup();
    }
    ImGui::EndPopup();
}

4. Widgets Básicos

Widgets de entrada

ImGui ofrece diversos widgets para interactuar con el usuario:

Slider

float value = 0.5f;
ImGui::SliderFloat("Deslizador", &value, 0.0f, 1.0f);

Checkbox

bool enabled = true;
ImGui::Checkbox("Habilitado", &enabled);

Input Text

char text[128] = "Hola";
ImGui::InputText("Texto", text, IM_ARRAYSIZE(text));

Combo Box

const char* items[] = {"Opción 1", "Opción 2", "Opción 3"};
static int item_current = 0;
ImGui::Combo("Combo", &item_current, items, IM_ARRAYSIZE(items));

Widgets de visualización

Elementos para mostrar información:

// Texto simple
ImGui::Text("Texto estático");

// Texto formateado
ImGui::Text("FPS: %.1f", io.Framerate);

// Etiqueta + valor en la misma línea
ImGui::LabelText("Etiqueta", "Valor");

// Barra de progreso
ImGui::ProgressBar(0.75f, ImVec2(-FLT_MIN, 0), "75%");

// Separador visual
ImGui::Separator();

// Bullet points
ImGui::BulletText("Item 1");
ImGui::BulletText("Item 2");

Widgets de organización

Para agrupar y organizar elementos:

// Grupos con bordes
ImGui::BeginGroup();
ImGui::Text("Grupo 1");
ImGui::Button("Botón A");
ImGui::EndGroup();

ImGui::SameLine();

ImGui::BeginGroup();
ImGui::Text("Grupo 2");
ImGui::Button("Botón B");
ImGui::EndGroup();

// Árboles y secciones plegables
if (ImGui::TreeNode("Sección avanzada")) {
    ImGui::Text("Contenido oculto...");
    ImGui::TreePop();
}

// Pestañas
if (ImGui::BeginTabBar("MisPestañas")) {
    if (ImGui::BeginTabItem("General")) {
        ImGui::Text("Contenido pestaña General");
        ImGui::EndTabItem();
    }
    if (ImGui::BeginTabItem("Avanzado")) {
        ImGui::Text("Contenido pestaña Avanzado");
        ImGui::EndTabItem();
    }
    ImGui::EndTabBar();
}

¡Cuidado con los IDs!

ImGui usa los textos de los widgets como identificadores. Si necesitas varios widgets con el mismo texto, usa ## para añadir un identificador único:

ImGui::Button("Aceptar##1");  // ID único: "Aceptar##1"
ImGui::Button("Aceptar##2");  // ID único: "Aceptar##2"

5. Diseño y Organización

Posicionamiento de elementos

Controla cómo se disponen los widgets en pantalla:

// Elementos en la misma línea
ImGui::Button("Botón 1");
ImGui::SameLine();
ImGui::Button("Botón 2");

// Espaciado
ImGui::Spacing();  // Espacio vertical
ImGui::SameLine(0, 20);  // Espacio horizontal de 20px

// Posicionamiento absoluto
ImGui::SetCursorPos(ImVec2(100, 50));
ImGui::Text("Posicionado en (100,50)");

// Centrado
ImGui::SetCursorPosX((ImGui::GetWindowSize().x - ImGui::CalcTextSize("Texto centrado").x) * 0.5f);
ImGui::Text("Texto centrado");

Tamaños y restricciones

Controla el tamaño de los widgets y ventanas:

// Tamaño fijo para un botón
ImGui::Button("Botón Grande", ImVec2(200, 50));

// Ventana con tamaño inicial
ImGui::SetNextWindowSize(ImVec2(400, 300), ImGuiCond_FirstUseEver);
ImGui::Begin("Ventana con tamaño personalizado");

// Widget que ocupa todo el ancho disponible
ImGui::Button("Botón ancho completo", ImVec2(-FLT_MIN, 0));

Columnas

Crea diseños multi-columna:

// Diseño con 2 columnas
ImGui::Columns(2, "misColumnas", true);
ImGui::Text("Columna 1");
ImGui::NextColumn();
ImGui::Text("Columna 2");
ImGui::Columns(1);  // Vuelve a 1 columna

Docking (acoplamiento)

Para interfaces más complejas con ventanas acoplables:

// Habilita el espacio de docking
ImGui::DockSpaceOverViewport(ImGui::GetMainViewport());

// Ventana acoplable
ImGui::Begin("Ventana Acoplable");
ImGui::Text("Puedes arrastrar esta ventana para acoplarla");
ImGui::End();

Consejo de diseño

Usa ImGui::BeginChild() y ImGui::EndChild() para crear áreas desplazables independientes dentro de tus ventanas:

ImGui::BeginChild("PanelDesplazable", ImVec2(0, 200), true);
// Contenido largo aquí...
ImGui::EndChild();

6. Personalización y Estilos

Estilos básicos

Puedes personalizar casi todos los aspectos visuales de ImGui:

// Cambiar el estilo por defecto
ImGuiStyle& style = ImGui::GetStyle();
style.WindowRounding = 5.0f;  // Esquinas redondeadas
style.FramePadding = ImVec2(10, 5);  // Más espacio en botones
style.Colors[ImGuiCol_Button] = ImVec4(0.26f, 0.59f, 0.98f, 0.40f);  // Color de botón

// Temas predefinidos
ImGui::StyleColorsDark();    // Tema oscuro (por defecto)
ImGui::StyleColorsLight();   // Tema claro
ImGui::StyleColorsClassic(); // Tema clásico (similar a imgui 1.0)

Colores personalizados

Puedes modificar todos los colores de la interfaz:

// Modificar colores específicos
ImVec4* colors = ImGui::GetStyle().Colors;
colors[ImGuiCol_Text] = ImVec4(1.00f, 1.00f, 1.00f, 1.00f);
colors[ImGuiCol_WindowBg] = ImVec4(0.06f, 0.06f, 0.06f, 0.94f);
colors[ImGuiCol_Button] = ImVec4(0.44f, 0.44f, 0.44f, 0.40f);
colors[ImGuiCol_ButtonHovered] = ImVec4(0.46f, 0.47f, 0.48f, 1.00f);
colors[ImGuiCol_ButtonActive] = ImVec4(0.42f, 0.42f, 0.42f, 1.00f);

Fuentes personalizadas

Puedes cargar y usar diferentes fuentes:

// Cargar una fuente de mayor tamaño
ImGuiIO& io = ImGui::GetIO();
io.Fonts->AddFontFromFileTTF("arial.ttf", 18.0f);

// Usar una fuente diferente para un texto específico
ImFont* bold_font = io.Fonts->AddFontFromFileTTF("arialbd.ttf", 14.0f);
ImGui::PushFont(bold_font);
ImGui::Text("Este texto está en negrita");
ImGui::PopFont();

Herramientas útiles

  • ImGui Style Editor: Usa ImGui::ShowStyleEditor() para ajustar estilos interactivamente.
  • Generadores de temas: Herramientas online como ImThemes pueden ayudarte a crear esquemas de color.
  • Font Awesome: Puedes integrar iconos con IconFontCppHeaders.

Estilos por widget

Puedes modificar el estilo de widgets individuales:

// Botón con estilo personalizado
ImGui::PushStyleColor(ImGuiCol_Button, ImVec4(1, 0, 0, 1));  // Botón rojo
ImGui::PushStyleVar(ImGuiStyleVar_FrameRounding, 10.0f);     // Esquinas redondeadas
if (ImGui::Button("Botón especial")) {
    // Acción del botón
}
ImGui::PopStyleVar();
ImGui::PopStyleColor();

7. Temas Avanzados

Widgets personalizados

Puedes crear tus propios widgets usando las primitivas de dibujo:

// Widget de barra de progreso circular personalizado
void DrawCircularProgress(float progress, float radius) {
    ImVec2 center = ImGui::GetCursorScreenPos();
    center.x += radius;
    center.y += radius;
    
    // Fondo del círculo
    ImGui::GetWindowDrawList()->AddCircle(center, radius, IM_COL32(100, 100, 100, 255), 0, 3.0f);
    
    // Parte completada
    ImGui::GetWindowDrawList()->PathArcTo(center, radius, -IM_PI/2, -IM_PI/2 + 2*IM_PI*progress, 32);
    ImGui::GetWindowDrawList()->PathStroke(IM_COL32(0, 255, 0, 255), false, 3.0f);
    
    // Texto en el centro
    char text[32];
    sprintf(text, "%.0f%%", progress*100);
    ImVec2 text_size = ImGui::CalcTextSize(text);
    ImGui::GetWindowDrawList()->AddText(
        ImVec2(center.x - text_size.x/2, center.y - text_size.y/2),
        IM_COL32(255, 255, 255, 255), text);
    
    // Ajusta el cursor para el siguiente widget
    ImGui::Dummy(ImVec2(radius*2, radius*2));
}

// Uso:
DrawCircularProgress(0.75f, 30.0f);

Optimización

Consejos para mantener tu interfaz eficiente:

Mejores prácticas

  • Evita recrear datos: Carga texturas y fuentes una vez al inicio.
  • Usa clippings: Para listas largas, usa ImGuiListClipper.
  • Reduce widgets ocultos: No crees widgets que no se van a mostrar.
  • Cachea cálculos costosos: Si un valor no cambia entre frames, no lo recalcules.

Extensiones populares

La comunidad ha creado muchas extensiones útiles:

Extensión Descripción Enlace
ImPlot Gráficos y visualización de datos GitHub
ImNodes Sistema de nodos y conexiones GitHub
ImGuiColorTextEdit Editor de código con resaltado de sintaxis GitHub
ImGuizmo Gizmos para manipulación 3D GitHub

Serialización del estado

Puedes guardar y cargar el estado de la interfaz:

// Guardar el estado
ImGui::SaveIniSettingsToDisk("imgui.ini");

// Cargar el estado
ImGui::LoadIniSettingsFromDisk("imgui.ini");

// También puedes usar strings en memoria
std::string ini_data = ImGui::SaveIniSettingsToMemory();

Ejemplos Prácticos

Demo integrada

ImGui incluye una demo completa que muestra todas sus capacidades. Para usarla:

#include "imgui.h"
#include "imgui_impl_glfw.h"
#include "imgui_impl_opengl3.h"

// En tu bucle principal
bool show_demo_window = true;
ImGui::ShowDemoWindow(&show_demo_window);

Ventana de demostración de Dear ImGui mostrando varios widgets y características.

Ejemplo básico completo

Aquí tienes un ejemplo mínimo funcional con GLFW y OpenGL:

main.cpp 
#include "imgui.h"
#include "imgui_impl_glfw.h"
#include "imgui_impl_opengl3.h"
#include <GLFW/glfw3.h>

int main() {
    // Inicializa GLFW
    if (!glfwInit()) return 1;
    
    // Crea ventana
    GLFWwindow* window = glfwCreateWindow(1280, 720, "ImGui Example", NULL, NULL);
    if (!window) {
        glfwTerminate();
        return 1;
    }
    glfwMakeContextCurrent(window);
    
    // Inicializa ImGui
    IMGUI_CHECKVERSION();
    ImGui::CreateContext();
    ImGuiIO& io = ImGui::GetIO(); (void)io;
    
    // Configura estilo
    ImGui::StyleColorsDark();
    
    // Inicializa backends
    ImGui_ImplGlfw_InitForOpenGL(window, true);
    ImGui_ImplOpenGL3_Init("#version 130");
    
    // Variables de estado
    bool show_window = true;
    float slider_value = 0.5f;
    
    // Bucle principal
    while (!glfwWindowShouldClose(window)) {
        glfwPollEvents();
        
        // Limpia pantalla
        glClear(GL_COLOR_BUFFER_BIT);
        
        // Nueva frame ImGui
        ImGui_ImplOpenGL3_NewFrame();
        ImGui_ImplGlfw_NewFrame();
        ImGui::NewFrame();
        
        // Crea tu interfaz
        if (show_window) {
            ImGui::Begin("Ventana de Ejemplo", &show_window);
            ImGui::Text("¡Hola, mundo!");
            ImGui::SliderFloat("Valor", &slider_value, 0.0f, 1.0f);
            if (ImGui::Button("Cerrar")) {
                show_window = false;
            }
            ImGui::End();
        }
        
        // Renderiza ImGui
        ImGui::Render();
        ImGui_ImplOpenGL3_RenderDrawData(ImGui::GetDrawData());
        
        // Intercambia buffers
        glfwSwapBuffers(window);
    }
    
    // Limpieza
    ImGui_ImplOpenGL3_Shutdown();
    ImGui_ImplGlfw_Shutdown();
    ImGui::DestroyContext();
    
    glfwDestroyWindow(window);
    glfwTerminate();
    
    return 0;
}
